사용자는 방명록에 글을 작성하고 댓글을 남길 수 있으며,TailwindCSS를 이용한 반응형 UI로 다양한 디바이스에서도 안정적인 사용이 가능합니다.

## 🔧 기술 스택

### Frontend

- **React**
- **Next.js**
- **TypeScript**
- **TailwindCSS**

### Backend

- **Node.js**
- **NestJS**
- **TypeORM + MySQL**
- **JWT 기반 인증**
